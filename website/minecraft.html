Server Rules

  1. Treat other players and community members with respect.
  2. Do not use hacked clients, prohibited modifications, exploits, duplication glitches, or other unfair advantages.
  3. Do not damage, steal from, or interfere with protected towns outside of permitted server mechanics.
  4. Towns and nations must follow all rules involving claims, warfare, diplomacy, and protected areas.
  5. Do not intentionally create machines, farms, or entities that cause unreasonable server lag.
  6. Do not evade punishments or assist another player in avoiding moderation.

These rules are placeholders and may change before the server opens.

Plugins and Features

Paper

Provides the main server software along with improved performance, stability, configuration options, and plugin support.

Towny

Allows players to create towns, claim land, join nations, manage residents, and build protected communities.

SiegeWar

Adds structured warfare between towns and nations while working alongside Towny's land and nation systems.

The New Economy

Provides the server economy and supports the custom currency used for trading and town finances.

QuickShop

Allows players to create chest shops and participate in a player-driven economy.

Geyser

Allows Bedrock Edition players to connect and play alongside Java Edition players.

ViaVersion and ViaBackwards

Expands client compatibility while allowing the server to remain on a stable base version.

BlueMap

Creates an interactive three-dimensional online map of the server world.
